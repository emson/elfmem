# Implementation Plans

These are the design records written before each subsystem was built. They are
**frozen artifacts** — they describe the intent and trade-offs at the time of
implementation, not the current state of the code.

Read them to understand *why* a subsystem works the way it does. For the current
behaviour, read the source code and `docs/amgs_architecture.md`.
